║                          📊 COMMANDES UTILES                                  ║
╚══════════════════════════════════════════════════════════════════════════════╝

🔹 PM2 (Gestion des processus)
   pm2 status                    # Statut des applications
   pm2 logs                      # Tous les logs
   pm2 logs visualworld-api      # Logs API
   pm2 logs visualworld-front    # Logs Frontend
   pm2 restart all               # Redémarrer tout
   pm2 monit                     # Monitoring temps réel

🔹 Git (Workflow)
   git checkout dev              # Aller sur dev
   git push origin dev           # Push et déploiement auto
   
   git checkout test             # Aller sur test
   git merge dev                 # Merger dev dans test
   git push origin test          # Push et déploiement auto
   
   git checkout main             # Aller sur prod
   git merge test                # Merger test dans prod
   git push origin main          # Push et déploiement auto

🔹 Monitoring
   pm2 status                    # Statut PM2
   systemctl status nginx        # Statut Nginx
   ss -tlnp | grep 3333          # Port Frontend
   ss -tlnp | grep 8080          # Port API

🔹 Logs
   tail -f /var/log/visualworld-api-out.log
   tail -f /var/log/visualworld-front-out.log
